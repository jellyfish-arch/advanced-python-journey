# 📊 Data Structures & Algorithms

Core data structures and algorithms implemented from scratch in Python. Each program is standalone and includes demonstrations with sample data.

## 📁 Programs

| #  | File | Topic | Key Concepts |
|----|------|-------|--------------|
| 01 | `01_linked_list.py` | Singly Linked List | Node-based structure, insert/delete/search, in-place reversal |
| 02 | `02_stack.py` | Stack (LIFO) | Push/pop/peek, balanced parentheses checker, string reversal |
| 03 | `03_queue.py` | Queue & Circular Queue | FIFO operations, ring buffer, round-robin task scheduler |
| 04 | `04_binary_search_tree.py` | Binary Search Tree | Insert/search/delete, in/pre/post/level-order traversals |
| 05 | `05_hash_table.py` | Hash Table | Separate chaining, dynamic resizing, O(1) average lookup |
| 06 | `06_sorting_algorithms.py` | Sorting Algorithms | Bubble, Selection, Insertion, Merge, Quick Sort + benchmarks |
| 07 | `07_searching_algorithms.py` | Searching Algorithms | Linear, Binary (iterative & recursive), Jump Search |
| 08 | `08_graph_bfs_dfs.py` | Graph Traversals | Adjacency list, BFS, DFS, shortest path (unweighted) |
| 09 | `09_heap_priority_queue.py` | Min-Heap & Priority Queue | Sift-up/down, heapify, priority-based task scheduling |
| 10 | `10_dynamic_programming.py` | Dynamic Programming | Fibonacci, 0/1 Knapsack, LCS, Coin Change |

## 🚀 How to Run

Each script is self-contained with a demo in its `__main__` block:

```bash
python skills/dsa/01_linked_list.py
python skills/dsa/06_sorting_algorithms.py
```

## 📌 Focus

- Building data structures from scratch (no built-in shortcuts)
- Understanding time & space complexity trade-offs
- Applying DSA to real-world problems (scheduling, pathfinding, optimization)
- Writing clean, well-documented, Pythonic implementations
