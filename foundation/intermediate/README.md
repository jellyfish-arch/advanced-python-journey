# 🐍 Intermediate Python

Where real Python starts. These programs cover the core intermediate concepts every developer needs to write clean, efficient, and professional Python code.

## Programs

| #  | File | Topic | Key Concepts |
|----|------|-------|--------------|
| 01 | `01_list_comprehensions.py` | List Comprehensions | Basic, nested, conditional, dict & set comprehensions |
| 02 | `02_lambda_map_filter.py` | Lambda, Map, Filter | Anonymous functions, `map()`, `filter()`, `reduce()`, `sorted()` |
| 03 | `03_decorators_intro.py` | Decorators (Intro) | `@wraps`, timer, logger, auth, stacking decorators |
| 04 | `04_error_handling.py` | Error Handling | `try/except/else/finally`, custom exceptions, chaining |
| 05 | `05_args_kwargs.py` | *args & **kwargs | Flexible args, unpacking, forwarding, keyword-only |
| 06 | `06_string_formatting.py` | String Formatting | f-strings, `format()`, `Template`, regex basics |
| 07 | `07_file_io.py` | File I/O | Text, JSON, CSV, `pathlib`, context managers |
| 08 | `08_iterators_generators.py` | Iterators & Generators | Custom iterators, `yield`, `send()`, pipelines, memory |
| 09 | `09_collections_module.py` | Collections Module | `Counter`, `defaultdict`, `namedtuple`, `deque`, `ChainMap` |
| 10 | `10_closures_first_class.py` | Closures & First-Class Functions | Higher-order functions, factories, memoization |

## How to Run

```bash
python foundation/intermediate/01_list_comprehensions.py
```

Each program is self-contained with a `if __name__ == "__main__"` block and produces readable terminal output.
