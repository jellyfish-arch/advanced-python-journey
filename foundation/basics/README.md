# 🐍 Python Console Programs Collection

This repository contains a set of basic Python console-based programs created for practice and consistency.
The goal is to build logic, improve coding habits, and maintain a daily GitHub streak.

---

## 📌 Programs Included

1. **Calculator**

   * Performs basic arithmetic operations (+, -, *, /)

2. **Number Guessing Game**

   * User guesses a randomly generated number

3. **Password Generator**

   * Generates secure random passwords

4. **To-Do List**

   * Add and view tasks using a simple menu

5. **Word Counter**

   * Counts words and characters in input text

6. **File Organizer**

   * Renames `.txt` files automatically

7. **Expense Tracker**

   * Tracks and sums user expenses

8. **Login System**

   * Basic username and password validation

9. **URL Status Checker**

   * Checks HTTP status of a given URL

10. **Fibonacci Generator**

* Prints Fibonacci sequence up to n terms

---

## ⚙️ Requirements

* Python 3.x
* `requests` library (only for URL Checker)

Install dependency:

```bash
pip install requests
```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to the project folder:

```bash
cd your-repo-name
```

3. Run any file:

```bash
python filename.py
```
